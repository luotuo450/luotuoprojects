1.HTML介绍

  • HTML是用来描述网页的一种语言
  • HTML是Hyper Text Markup Language的缩写,意思是超文本标记语言
  • HTML不是一种编程语言,而是一种标记语言
  • HTML使用标记标签来描述网页
  • HTML标签是成对出现的,有:<开始标签>和</结束标签>
  • HTML文件的后缀是.htm或.html,两者之间没有区别

2.使用VScode编写html

注意：如果你的电脑是Windows7以前(包括Windows7),就无法下载vscode，建议直接使用电脑自带的记事本来编写HTML文件

  1. 打开VScode
  2. 新建一个文件夹
  3. 在文件夹中新建一个文件,文件名是index.html,(当你打开浏览器时,如果只输入文件夹路径,浏览器就会自动定位到文件夹中的index.html)
  4. 在文件中输入html时,会弹出一个代码提示框
  5. 选择html:5,然后按回车键,就会自动生成一个html模版(下一章讲解模版)

开始愉快地写代码吧

图2.1.1 VScode弹出代码提示框

选择VScode作为HTML编辑器的好处就是里面有许多插件,下面将给大家介绍4款插件

  1. Auto Close Tag:顾名思义,就是当你输入开始标签时，这款插件会帮你自动补全结束标签
  2. Auto Rename Tag:当你修改了开始(结束)标签的名称时，这个插件会自动修改结束(开始)标签的名称
  3. Live Preview:不需要打开浏览器,在VScode中打开HTML文件,点击右上角的图标就可实时预览HTML文件在网页中显示的样子,效果如下
  4. open in browser:有时候我们的HTML必须用浏览器打开,比如说你写的网页中有一个链接跳转到www.baidu.com,由于www.baidu.com不属于你的本地文件,所以上一款插件也无能为力,这款插件只需要你先保存文件,然后在你编写HTML文件的窗口右键,选择open in default browser,就能直接在你的默认浏览器中打开HTML文件

3.使用记事本编写html

  • 在桌面上右键
  • 选择新建
  • 选择文本文档
  • 将文件名的后缀改成.html或.htm
  • 对着该文件右键
  • 选择用记事本打开该文件

最后,你就可以开始愉快地写代码了

4.html模版

由于有些人的电脑无法下载vscode,我这里把vscode生成的模版给大家放出来,给大家逐行解析

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Document</title>
</head>
<body>

</body>
</html>

首先,我们看第一行,<!DOCTYPE html>是使用最新版HTML的文档声明,目前最新版是HTML5

第二行<html lang="en">是HTML5的根标签,html的所有内容都要写在<html></html>标签里,lang属性是用来指定当前文档的语言,en是英文,zh-CN是中文,其实这个lang属性是给搜索引擎判断你的网页是中文还是英文,以便用户找到适合自己阅读的网站,所以如果你写的网站不是纯英文的话,把这里改成<html lang="zh-CN">

第三行是<head>,里面写的是网页的头部信息,比如编码格式什么的

第四行 <meta charset="UTF-8"> 是HTML5的meta标签,charset属性是用来指定当前文档的字符集,UTF-8是包含所有语言的编码,如果没有什么特殊需求都用UTF-8

第五行: 1、name属性值viewport表示视口，不仅匹配PC端，也可匹配移动端。
2、content属性值：
（1）width：可视区域的宽度，值可为数字或关键词
（2）device-width:设备屏幕的物理宽度
（3）intial-scale:页面首次被显示是可视区域的缩放级别，取值1.0则页面按实际尺寸显示，无任何缩放
扩展:
height:可视区域的高度
maximum-scale:允许用户缩放的最大比例
minimum-scale:允许用户缩放的最小比例
user-scalable:是否允许用户缩放,no 表示禁止缩放,yes代表允许

第六行是<title>Document</title>,title标签是用来指定当前页面的标题,这个标题会显示在浏览器的标题栏里,比如百度首页的标题就是

第七行是</head>,表示头部信息结束

第八行是<body>,表示网页的主体部分

第十行是</body>,表示网页的主体部分结束

第十一行是</html>,表示整个网页结束

注意:
1、标签名和属性值尽量小写
2、标签尽量正确关闭
3、标签尽量正确嵌套
我说尽量是因为即使你不这样做浏览器也能正确解析,但是在XHTML中是不符合规范的,养成良好的写代码习惯总是不会有坏处的

5.HTML段落

在HTML中,最常见也是最普通的输出标签就是<p>标签,里面的文字正常输出,但是里面的HTML标签会被解析

你可以在你的HTML文件的body标签里输入<p>hello world</p>,然后保存文件,在浏览器中打开该文件(或者使用VScode插件Live Preview),然后你就能在界面中看到hello world,您也可以亲自试一试
